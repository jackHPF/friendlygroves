# Hero Background Image

The homepage hero section uses a coastal landscape image as the background.

## Image Location
Place your coastal landscape image at:
```
public/images/coastal-landscape.jpg
```

## Supported Formats
- `.jpg` or `.jpeg` (recommended)
- `.png`
- `.webp`

## Image Requirements
- **Recommended size:** 1920x1080px or larger
- **Aspect ratio:** 16:9 works best
- **File size:** Keep under 2MB for fast loading
- **Content:** Coastal landscape with beach, ocean, and hills

## How to Add
1. Save your coastal landscape image
2. Rename it to `coastal-landscape.jpg` (or update the path in `components/Hero.tsx`)
3. Place it in the `public/images/` directory
4. The image will automatically appear as the hero background

## Alternative: Use Existing Image
If you have an existing image you'd like to use, you can:
1. Rename it to `coastal-landscape.jpg`, OR
2. Update the image path in `components/Hero.tsx` line 28
